# RSS AMSU Forward Operator

Python package for the RSS AMSU forward operator utilities, including table-based brightness temperature calculations and ERA5 helpers.

## Layout

- `rss_amsu_forward_operator/` - package source
- `examples/` - example scripts

## Quick Start

Run the example from the module folder:

```bash
cd RSS_AMSU_forward_operator
python examples/test_AMSU_forward_table.py
```
